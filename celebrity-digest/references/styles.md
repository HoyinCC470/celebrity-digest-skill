# 选定日报风格：极致紧凑型 (Extreme Compact Layout) - 终版极简 (标题+总结+链接)

经过对比评估，我们将详情页中原本展示的“代表帖正文（通常带有微博乱码和零散词句）”替换为由 **Agent 核心大脑整理的“AI 一句话极简总结”**。这极大地提升了日报的文字质量和专业观感，同时保留了直接跳转热度微博的直链。

## 终版优化要点

1. **今日动态总结仅保留标题**：`🎯 今日动态总结` 仅展示核心事件列表（如 `* 《开始推理吧》第二案上线与高光花絮`），不包含冒号和后置描述。
2. **用“一句话总结”替换“微博正文”**：
   - 彻底舍弃可能包含话题标签、空白和语气词的原始微博文本。
   - 替换为 1 行极简的 **“AI 一句话总结”**，字数严格控制在 **30 字左右**（通常为 28-35 字，不要过短。不带“总结：”前缀，且完全去除“工作室”、“视频组”、“后援会”、“站姐”等角色和账号前缀。措辞上必须使用热情激动的饭圈风格，融入情绪词汇）。
3. **移除“合并数”及皇冠标签**：标题末尾不再附带任何数量后缀，也不带皇冠 Emoji。
4. **三行极简结构 (3-line structure)**：每个详情项目下方包含三行 Bullet Points：
   - 第一行：`* ` 引导的一句话极简总结（由 Agent 总结出该主题的动态核心，直接以动作/事实开头，字数在 30 字左右，风格偏向热情、激动的饭圈话术）。
   - 第二行：`* 🔗 ` 引导的微博正文直达链接。
   - 第三行：`* 🔁 [转发] / 💬 [评论] / ❤️ [点赞] ` 引导的互动数据，独立成行，不使用任何括号。

---

## 终版极简排版规范

每一个详情动态统一使用以下结构规范：

```markdown
### [序号]. [Emoji] [主题/话题名称]
* [AI一句话极简事实总结，30字左右，无“总结：”前缀，完全去除“工作室”、“视频组”等前缀，措辞热情且偏向饭圈话术]
* 🔗 [微博正文](链接)
* 🔁 转发数 / 💬 评论数 / ❤️ 点赞数
```
